# 扫地机器人智能客服 Agent 项目

## 项目概述

本项目是一个基于 LangGraph + LangChain 构建的智能客服 Agent，专门为扫地机器人/扫拖一体机器人提供专业的问题解答和使用报告生成服务。Agent 采用 **ReAct（Reasoning + Acting）模式**，具备自主思考和工具调用的能力。

项目包含两个版本的 Agent 实现：
- **LangGraphReactAgent**: 基于 LangGraph StateGraph 的新版实现（推荐）
- **ReactAgent**: 基于 LangChain AgentBuilder 的旧版实现

## 核心组件详解

### 1. Agent 层

#### LangGraphReactAgent (推荐使用)

基于 LangGraph 的 `StateGraph` 构建，显式定义了节点和边，流程更清晰可控。

```python
class AgentState(TypedDict):
    messages: list  # 消息历史
    report: bool    # 是否报告生成模式
```

**节点定义:**

| 节点 | 函数 | 功能 |
|------|------|------|
| `model` | `model_node()` | 调用 LLM 生成回复，根据 `report` 状态切换不同 Prompt |
| `tools` | `tools_node()` | 执行模型请求的工具调用 |
| 条件边 | `should_continue()` | 判断是否继续工具调用循环，最多5次 |

#### ReactAgent (旧版)

基于 LangChain AgentBuilder 构建，使用中间件模式处理横切关注点。

---

### 2. 工具层 (Tools)

| 工具名称 | 入参 | 功能 | 使用场景 |
|---------|------|------|---------|
| `rag_summarize` | query: str | 从向量库检索相关资料并总结 | 需要专业产品知识时 |
| `get_weather` | city: str | 获取指定城市天气 | 环境适配咨询 |
| `get_user_location` | 无 | 获取用户所在城市 | 配合天气工具使用 |
| `get_user_id` | 无 | 获取用户唯一标识 | 生成个性化报告 |
| `get_current_month` | 无 | 获取当前月份 | 报告时间范围 |
| `fetch_external_data` | user_id, month | 获取用户使用记录 | 报告数据来源 |
| `fill_context_for_report` | 无 | 触发报告模式上下文注入 | **报告生成必调前置工具** |

---

### 3. 中间件层 (Middleware)

| 中间件 | 装饰器 | 功能 |
|-------|--------|------|
| `monitor_tool` | `@wrap_tool_call` | 工具执行监控、错误捕获、设置 report 标志 |
| `log_before_model` | `@before_model` | 模型执行前日志记录 |
| `report_prompt_switch` | `@dynamic_prompt` | 动态切换 Prompt（普通/报告模式）|

---

### 4. RAG 服务层

```
用户查询 ──▶ Retriever(k=3) ──▶ ChromaDB 向量检索
                              │
                              ▼
                    ┌─────────────────┐
                    │  PromptTemplate │
                    │  + Context 组装 │
                    └────────┬────────┘
                             │
                             ▼
                    ┌─────────────────┐
                    │  ChatTongyi     │
                    │  LLM 总结生成   │
                    └────────┬────────┘
                             │
                             ▼
                        最终回答
```

**向量存储特性:**
- 支持 PDF、TXT 格式文档
- 基于 MD5 去重，避免重复加载
- 文档分片 (chunk_size=200, overlap=20)
- k=3 检索返回最相关的3条文档

---

### 5. 模型层

| 模型 | 类型 | 用途 |
|------|------|------|
| `qwen3-max-2026-01-23` | ChatTongyi | 对话生成、工具调用决策 |
| `text-embedding-v4` | DashScope | 文档向量化、语义检索 |

## 报告生成模式

当用户请求生成报告时，Agent 进入特殊的报告生成流程：

```
用户: "生成我的使用报告"
         │
         ▼
┌─────────────────────────────────────────────────────┐
│  报告生成固定流程 (强约束)                          │
│                                                     │
│  1. get_user_id()      → 获取用户ID                │
│  2. get_current_month() → 获取当前月份              │
│  3. fill_context_for_report() → 触发上下文注入 ⚠️   │
│  4. fetch_external_data(user_id, month) → 获取数据 │
│                                                     │
└─────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────┐
│  中间件检测到 fill_context_for_report 被调用         │
│  → 设置 context["report"] = True                    │
│  → 后续 Prompt 切换为 report_prompt.txt             │
└─────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────┐
│  LLM 基于 report_prompt 生成格式规范的报告           │
│  包含: 清洁效率、耗材状态、使用对比、保养建议         │
└─────────────────────────────────────────────────────┘
```

## 配置说明

### agent.yml
```yaml
external_data_path: data/external/records.csv  # 用户外部数据路径
```

### rag.yml
```yaml
chat_model_name: qwen3-max-2026-01-23        # 对话模型名称
embedding_model_name: text-embedding-v4      # Embedding 模型名称
```

### chroma.yml
```yaml
collection_name: agent                        # 向量集合名
persist_directory: chroma_db                 # 持久化目录
k: 3                                         # 检索返回数量
chunk_size: 200                              # 分片大小
chunk_overlap: 20                            # 分片重叠
data_path: data                               # 知识库路径
allow_knowledge_file_type: ["txt", "pdf"]    # 支持的文件类型
```

## 使用示例

### 1. 基础问答
```python
from agent.langgraph_agent import LangGraphReactAgent

agent = LangGraphReactAgent()
for chunk in agent.execute_stream("扫地机器人主刷如何保养？"):
    print(chunk, end="", flush=True)
```

### 2. 报告生成
```python
agent = LangGraphReactAgent()
for chunk in agent.execute_stream("生成我的使用报告"):
    print(chunk, end="", flush=True)
```
